**Webcastradio4 ist ein Standard 3 Spalten Layout inkl. Slider, für ilch 2.0**

**Besonderheiten:**

Auf der Startseite wird direkt unter dem Slider der Radiostatus angezeigt
Der Radiostatus ist optimiert für einen laut.fm und einen SHOUTcast 2 Stream.
Im Footer ist auch noch eine navbar.

Der Radiostatus kann über Colorpicker im Admin farblich verändert werden.
Über den Admin kann man Moderatorenbilder uploaden, die im Radiostatus angezeigt werden.
Es werden auch die Coverbilder vom Titel angezeigt.
Es werden die Hörerzahlen und bei einem SHOUTcast auch noch die Bitrate angezeigt.
Zusätzlich ist eine Gruß- und Wunschbox integriert und auch über das Admin konfigurierbar.



**English:**

**Webcastradio4 is a standard 3-column layout with a slider, designed for ilch 2.0**

**Features:** 

The radio status is displayed on the homepage directly below the slider. The radio status is optimized for a laut.fm and a SHOUTcast 2 stream. There is also a navigation bar in the footer.
The color of the radio status can be changed via the color picker in the admin panel. Through the admin panel, you can upload host images that are displayed in the radio status. The cover art for the track is also displayed. The number of listeners is shown, and for a SHOUTcast stream, the bitrate is also displayed. Additionally, a greeting and request box is integrated and can be configured via the admin panel.
